**FindImplementorsTool.cs** — Tool description:
```
"Finds all types implementing an interface or extending a base class within an assembly. Use this when mapping the concrete implementations behind an abstraction, identifying plugin points, or understanding a polymorphic dispatch target. Returns paginated matches with type name, assembly, and direct-vs-transitive relationship."
```
Parameter descriptions:
- assemblyPath: `"Path to the .NET assembly (.dll/.exe)"`
- typeName: `"Full name of the interface or base class (e.g., 'System.IDisposable')"`
- maxResults: `"Maximum results to return (default 100)"`
- offset: `"Results to skip for pagination (default 0)"`

- assemblyPath: `"Path to the .NET assembly (.dll/.exe)"`
