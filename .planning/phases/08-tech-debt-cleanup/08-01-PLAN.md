public sealed class OutputDirectoryNotEmptyException : DomainException

- Aligns with mcp-tool-design SKILL.md Principle 8 ("Errors carry next-step hints") ‚Äî the message tells the agent what to do: "Specify an empty or non-existent directory."

- Add more properties (no `AssemblyPath`, no `EntryCount`) ‚Äî scope creep
- Introduce an `OutputDirectoryPath` value object ‚Äî explicitly deferred per CONTEXT.md
- Touch `Domain/Errors/MethodNotFoundException.cs` ‚Äî unrelated and explicitly out of scope per CONTEXT.md
