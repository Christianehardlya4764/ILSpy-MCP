<objective>
Add resolveDeep flag and verify/enhance inline metadata token resolution in IL disassembly tools.

Purpose: Agents reading IL output should see fully-qualified names with assembly scope for all metadata token operands (call, callvirt, newobj, ldfld, ldstr, etc.) without needing follow-up calls. The resolveDeep flag provides expanded resolution (full parameter signatures, expanded generics) for agents that need deeper detail. Per D-01: ReflectionDisassembler already does most resolution; the work is gap-filling and adding the resolveDeep enhancement layer.

Output: Updated IDisassemblyService interface, enhanced ILSpyDisassemblyService with resolveDeep post-processing, updated use cases and transport tools, new tests.
</objective>
